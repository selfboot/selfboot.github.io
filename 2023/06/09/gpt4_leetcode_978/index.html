这个题目叫做 “最长湍流子数组”，给定一个整数数组 arr，返回 arr 的最大湍流子数组的长度。一个子数组被称为湍流子数组，当且仅当在该子数组中，每一对相邻元素的比较符号在子数组中每次都会翻转。

更正式地说，一个子数组 [arr[i], arr[i + 1], ..., arr[j]] 被称为湍流子数组，当且仅当：

  • 对于 i <= k < j：
    • 当 k 是奇数时，arr[k] > arr[k + 1]
    • 当 k 是偶数时，arr[k] < arr[k + 1]
  • 或者，对于 i <= k < j：
    • 当 k 是偶数时，arr[k] > arr[k + 1]
    • 当 k 是奇数时，arr[k] < arr[k + 1]

约束条件：

  • 1 <= arr.length <= 4 * 10^4
  • 0 <= arr[i] <= 10^9
